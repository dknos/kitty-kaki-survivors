### Synthesized layers (original, MIT-licensed by this repo)
- **Crow caw** in `cinder_ambient.ogg` at t≈8 s and t≈28 s:
  FM-modulated sine pair (base 640 Hz / 580 Hz — distinctly lower than
  twilight's 780 Hz crow so the cinder crow reads as a different bird
  across the battlefield) with 22 Hz vibrato + 38 Hz tremolo for the
  rasping caw character. Two-burst envelope (0.32 s + 0.28 s, 120 ms gap)
  per call. Lowpassed at 1.7 kHz for distance smear. Generated by
  `python3` + numpy synthesis — no external sample used. Second instance
  also pitched 0.92x via `asetrate=40500` so the two crow calls don't
  sound identical.
- **Wood creak** in `cinder_ambient.ogg` at t≈15 s and t≈38 s:
  FM sine sweep 90 → 60 Hz with 3.5 Hz tremolo (rope-strain modulation)
  and a 1.8 s bell-shaped amplitude envelope. Lowpassed at 280 Hz so the
  creak sits low under the wind — the ruined siege engine settling on its
  broken counterweight. Original synth; the Kenney packs ship no rope/
  wood-strain sample. Second instance pitched 0.86x via `asetrate=38000`
  for variation.
- **Metal creak** in `cinder_ambient.ogg` at t≈4 s and t≈22 s:
  Detuned sine pair (380 Hz / 420 Hz with 6.5 Hz vibrato) over 1.4 s —
  corroded armature swaying in the wind. Lowpassed at 1.5 kHz. Original
  synth; provides the metallic "this stage was a war zone" punctuation
  the Kenney impact pack doesn't carry in a sustained form. Second
  instance pitched 1.04x via `asetrate=46000`.

## Processing

All assets processed with ffmpeg 7.0.2-static, encoded as Ogg Vorbis:
- **Ambient** normalized to **-22 LUFS** (matches forest + twilight
  ambient convention — ambient beds sit musically below SFX so the stage
  ambient gain at `0.55` in the audio bus reads cleanly when both
  procedural music and the bed are active. Note: the task brief said
  "normalize all to -16 LUFS" but the existing stage-ambient convention is
  -22 LUFS — followed the convention to avoid a 6 LU loudness slam when
  the player swaps stages. Verified against `assets/audio/forest/
  ATTRIBUTION.md` and `assets/audio/twilight/ATTRIBUTION.md`).
- **SFX** normalized to **-16 LUFS** (matches existing weapon SFX bus /
  EBU R128 / forest amber detonation + twilight fountain SFX).
- `cinder_ambient.ogg` is a **40 s seamless loop**: 45 s source rendered,
  last 5 s cross-faded onto the first 5 s (`acrossfade=d=5:c1=tri:c2=tri`),
  then concatenated with the [5, 40] body. Same recipe as forest +
  twilight ambient. Loop seam smoothness inherited from the 5 s tri-window
  crossfade.
- `ballista_repair_loop.ogg` is a **1.5 s seamless loop**: 1.6 s source
  rendered, last 0.1 s cross-faded onto the first 0.1 s, then concatenated
  with the [0.1, 1.5] body. Phase-2 Ballistas Agent calls
  `sfx.ballistaRepairLoop()` every ~1.4 s during the 10 s repair window —
  the slight overlap (~0.1 s) masks any residual seam and gives the
  continuous cranking feel. Implementation choice: normal SFX bucket (no
  special <audio loop=true> path) because the agent already tick()s; one
  less moving part in the audio bus and the 30 ms global throttle is well
  under the 1.4 s call cadence so repeated calls land cleanly.
- Ogg quality: `-q:a 2` for ambient (276 KB / 40 s ≈ 55 kbps — slightly
  above twilight's 138 KB / 40 s envelope because the cinder bed has more
  layered synth elements; still well inside the 800 KB ambient budget),
  `-q:a 4` for the short SFX so transients stay crisp.
