### Synthesized layers (original, MIT-licensed by this repo)
- **Crow caw** in `twilight_ambient.ogg` at t≈8s and t≈26s:
  FM-modulated sine pair (780 Hz / 720 Hz base) with 22 Hz vibrato + 38 Hz
  tremolo for the rasping caw character. Two-burst envelope (0.32s + 0.28s,
  120 ms gap) per call. Lowpassed (1.5-1.8 kHz) and stereo-delayed for
  distance. Generated by `python3` numpy synthesis — no external sample used.
- **Water trickle** in `fountain_pour.ogg`:
  Voss-McCartney pink noise (scipy `lfilter` with 4-tap IIR coefficients),
  bandpassed 180-3500 Hz, multiplied by a 7 Hz + 23 Hz tremolo envelope to
  emulate bubbling. Original synth — RPG-audio pack ships no water sound,
  the only Kenney path was to layer non-water samples and the result read as
  "rattling pebbles" not "water," so pink-noise tremolo was preferred for
  recognizability. 1.5 s, single-shot.
- **Swallow gulp** in `fountain_drink.ogg`:
  Three descending FM sine bursts (320 → 280 → 240 Hz base, each frequency-
  ramping 1.0x → 0.4x over 90-110 ms) representing throat-closing on each
  gulp. Original synth — Kenney packs don't ship a swallow sample. Layered
  under the `glass_001.ogg` chime tail above to land the magical-drink feel.

## Processing

All assets processed with ffmpeg 7.0.2-static, encoded as Ogg Vorbis:
- **Ambient** normalized to **-22 LUFS** (matches forest convention — ambient
  beds sit musically below SFX so the stage ambient gain at `0.55` in the
  audio bus reads cleanly when both procedural music and the bed are active).
- **SFX** normalized to **-16 LUFS** (matches existing weapon SFX bus / EBU R128
  / forest amber detonation + crystal shatter).
- `twilight_ambient.ogg` is a **40 s seamless loop**: 45 s source rendered,
  last 5 s cross-faded onto the first 5 s (`acrossfade=d=5:c1=tri:c2=tri`),
  then concatenated with the [5, 40] body. Same recipe as forest ambient.
  Loop seam smoothness verified by inspecting wrap region in the normalized
  WAV before encoding.
- Ogg quality: `-q:a 2` for ambient (138 KB / 40 s ≈ 27 kbps — matches forest
  ambient's 190 KB / 40 s envelope), `-q:a 4` for the short SFX so transients
  stay crisp.
